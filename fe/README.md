# Claro Frontend

A modern Next.js landing page for Claro, featuring a dark mode design with a beautiful meadow background.

## Getting Started

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.

### Production

```bash
npm run build
npm start
```

## Features

- Dark mode design
- Responsive layout
- Modern UI with smooth animations
- Optimized image loading with Next.js Image component
